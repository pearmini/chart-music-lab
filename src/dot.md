# Dot: Dodge Drum <a href="https://github.com/pearmini/chart-music-lab/blob/main/src/dot.md" target="_blank">&lt;/&gt;</a>

How do you make sounds from a dodge chart? Each dot is a drum. Click to play drums randomly generated by [Markov Chain](https://editor.p5js.org/luisa/sketches/ufHimtq6q), inspired by [Touch Pianist](https://touchpianist.com/).

```js
import {dot} from "./dot.js";
```

```js
const data = await FileAttachment("./data/cars.json").json();
```

```js
const drums = {
  hh: await FileAttachment("samples/hh.mp3").url(),
  hho: await FileAttachment("samples/hho.mp3").url(),
  kick: await FileAttachment("samples/kick.mp3").url(),
  snare: await FileAttachment("samples/snare.mp3").url(),
};
```

```js
const node = await dot(data, {width, drums});

invalidation.then(() => node.dispose());
```
